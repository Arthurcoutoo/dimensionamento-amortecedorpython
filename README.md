# Dimensionamento de Sistema de Amortecimento Automotivo

Este projeto apresenta uma rotina em Python desenvolvida para calcular os parâmetros dinâmicos e estruturais de um sistema de suspensão linear do tipo massa-mola-amortecedor.

## 🛠️ Tecnologias Utilizadas
* Python 3
* Biblioteca Math (Nativa)

## 📐 Fórmulas e Conceitos Físicos Aplicados
O algoritmo baseia-se na teoria de vibrações mecânicas:
* Rigidez da Mola: $k = m \cdot \omega_n^2$
* Amortecimento Crítico: $C_c = 2 \cdot m \cdot \omega_n$

## 📂 Organização do Repositório
* `main.py`: Código-fonte com a rotina de dimensionamento.
* `imagens/`: Pasta com as tomadas de decisão geradas por IA.
